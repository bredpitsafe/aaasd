### Getting started
* [Install The Rust Toolchain](https://www.rust-lang.org/tools/install)


* Add path to .zshrc ```/Users/USER_NAME/.cargo/bin```


* [Install wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)


* Build ```wasm-pack build```
